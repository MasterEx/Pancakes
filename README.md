Pancakes
========
An IRCbot and logger in pure Bash.

Deps
----
- Bash shell
- coreutils (/bin/{mkdir,mkfifo,rm}, /usr/bin/env)

Proposed features
-----------------
- respond to queries
  requires setting up a new fd

Proposed commands
-----------------
- !all
  hilight everyone in the room
